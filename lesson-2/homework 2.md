# Homework — Session 2

## Before Next Class

### 1. Make sure you have GitHub access
- [ ] Git installed (`git --version` should show a version number)
- [ ] GitHub CLI installed (`gh --version`)
- [ ] Authenticated with GitHub (`gh auth login` — use GitHub.com, HTTPS, login via browser)
- [ ] Shared your GitHub username in the #tmp-ai-builders-curriculum Slack channel

If you hit a wall with authentication or "command not found" errors, screenshot it and paste into Claude Code — it'll walk you through fixing it.

### 2. Review Ryan's Git slides
Ryan's interactive slide deck covers all four core concepts (commits, branches, push/pull, pull requests). Open `intro-to-git.html` in your browser and click through it at your own pace.

### 3. Keep building
Keep iterating on your project from Session 1. Use the same three methods:
- **Voice dump → /lfg** for full rebuilds
- **Voice dump → Claude Code** for quick passes
- **Agentation** for element-level feedback

### 4. Try Git with Claude Code
If you're feeling adventurous, try asking Claude Code to:
- "Initialize a git repo in this folder"
- "Create a new branch called my-experiment"
- "Show me the git status"
- "Commit my changes with a message describing what I did"

You don't need to push anything to GitHub yet — just get comfortable with the language.
